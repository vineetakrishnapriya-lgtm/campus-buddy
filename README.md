🎓 Campus Buddy
The Smart Digital Companion for College Campuses

Campus Buddy is an AI-powered campus management platform designed to simplify student life by centralizing academic resources, automating queries, and improving communication between students and administration.

It is built to solve common campus problems like repeated queries, scattered resources, infrastructure complaint tracking, and lack of student visibility.

🚀 Features
🤖 AI Campus Chatbot

24/7 smart assistant

Answers queries about exams, fees, contacts

Uses verified college documents only (RAG-based system)

Zero hallucination responses

🛠 Infrastructure Complaint System

Upload photo + room number

Track status: Pending → In Progress → Resolved

Transparent issue tracking

Improved accountability

📚 Academic Resource Hub

Admin-uploaded PDFs & videos

Subject-wise filtering

Organized & searchable content

Centralized repository

👥 Hackathon & Project Team Connect

👤 Student Profile Dashboard

GitHub & LinkedIn integration

Encourages skill development

🔐 Admin Dashboard

Manage complaints

How It Works

Admin uploads official college documents.

Documents are stored securely and embedded for AI retrieval.

Students ask questions through chatbot.

AI responds only using verified uploaded documents.

Complaints and resources are managed via dashboard.
